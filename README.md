# Sample Server

> Sample Express Template, ejs and ES6 enabled

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i Sample Server --save-dev
```

## Usage

```js
var Sample Server = require('Sample Server');
```

## Running tests

Install dev dependencies:

```sh
$ npm i -d && npm test
```
